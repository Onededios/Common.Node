## Installation

```bash
npm install @your-scope/common-lib
# or
pnpm add @your-scope/common-lib
# or
yarn add @your-scope/common-lib
```

### Requirements

- Node.js ≥ 18
- ESM compatible runtime (the package ships ESM + CommonJS builds).
